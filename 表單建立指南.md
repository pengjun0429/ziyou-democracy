# 資優民主國 國民入籍申請表 — 建立指南

## 表單基本設定

- 表單標題：資優民主國國民入籍申請表
- 表單說明：
  > 歡迎申請加入資優民主國！請如實填寫以下資料，送出後將由國家相關單位進行審核，審核通過後即成為本國正式國民。

## 分段與欄位（使用「前往區段」功能）

### 區段 1：基本資料

1. 姓名（簡答題）
2. 哪個國中？（簡答題）
   - 說明：例如「鳳甲國中」
3. 幾年級？（簡答題）
   - 說明：例如「8年級」
4. 班級（簡答題）
5. 是否為老師？（單選題）
   - 是 → 前往區段 4（申請理由）
   - 否 → 繼續（區段 2）

### 區段 2：身份確認

6. 是否代表整個資優班？（單選題）
   - 是 → 前往區段 4（申請理由）
   - 否 → 前往區段 3（補充資料）

### 區段 3：補充資料（非老師且非代表資優班者填寫）

7. 座號（簡答題）
8. 聯絡方式（簡答題）
   - 說明：請填寫可聯繫的 Email 或電話

### 區段 4：申請理由

9. 申請理由（段落）

## 設定建議

- 開啟「限制僅限回覆 1 次」（需登入 Google 帳號）
- 開啟「收集電子郵件地址」
- 允許回覆者「編輯回覆內容」

## 實作步驟

1. 建立表單後，新增 4 個區段（預設已有一個，再新增三個）
2. 將各欄位分別放入對應區段
3. 在「是否為老師？」點選右下角「…」→ 開啟「根據答案前往對應區段」，設定「是」→ 區段 4、「否」→ 區段 2
4. 在「是否代表整個資優班？」開啟「根據答案前往對應區段」，設定「是」→ 區段 4、「否」→ 區段 3

## 取得連結

建立完成後，按右上角「傳送」→ 複製表單網址。

## 自動回覆信

```javascript
function sendCustomEmail(e) {
  if (!e || !e.response) return;

  var responses = e.response.getItemResponses();
  var email = e.response.getRespondentEmail();
  var name = "", isTeacher = "", isRepresentative = "", school = "", grade = "";

  for (var i = 0; i < responses.length; i++) {
    var item = responses[i].getItem().getTitle();
    var answer = responses[i].getResponse();
    if (item.indexOf("姓名") !== -1) name = answer;
    if (item.indexOf("老師") !== -1) isTeacher = answer;
    if (item.indexOf("代表") !== -1) isRepresentative = answer;
    if (item.indexOf("國中") !== -1) school = answer;
    if (item.indexOf("年級") !== -1) grade = answer;
  }

  if (!email) return;

  var fullSchool = school + grade + "資優班";
  var subject, message;

  if (isTeacher === "是") {
    subject = "資優民主國入籍申請 - 教師審核通知";
    message = name + " 老師您好，\n\n感謝您申請加入資優民主國。您的教師身份申請將由專人審核，請耐心等候通知。\n\n審核通過後，將核發貴班「資優民主國班級身分證」及「班級護照」。\n\n資優民主國 入籍審核委員會";
  } else if (isRepresentative === "是") {
    subject = "資優民主國入籍申請 - 代表審核通知";
    message = name + " 您好，\n\n感謝您代表 " + fullSchool + " 申請加入資優民主國。我們將盡快審核您的代表資格與申請。\n\n審核通過後，將核發貴班「資優民主國班級身分證」及「班級護照」。\n\n資優民主國 入籍審核委員會";
  } else {
    subject = "資優民主國入籍申請 - 個人審核通知";
    message = name + " 同學您好，\n\n感謝您以 " + fullSchool + " 的身分申請加入資優民主國。您的個人入籍申請已收到，將由相關單位進行審核。\n\n審核通過後，將核發您的「資優民主國國民身分證」及「國民護照」。\n\n資優民主國 入籍審核委員會";
  }

  MailApp.sendEmail(email, subject, message);
}
```

### 設定步驟

1. 在表單編輯器，點右上角「…」→「指令碼編輯器」
2. 貼入上述程式碼，儲存
3. 左側「觸發條件」→「+ 新增觸發」
   - 函式：sendCustomEmail
   - 部署方式：上端
   - 事件來源：表單
   - 事件類型：提交表單時
4. 儲存並授權即可
