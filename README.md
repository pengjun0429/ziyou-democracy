# 資優民主國 - 微國家項目

## 項目概述
這是一個完整的微國家展示系統，包含靜態網站、文件管理系統、互動式國家儀表板以及微國家百科。

## 文件結構

```
資優民主國/
├── index.html              # 主頁面 - 資優民主國官方網站
├── micronation-wiki.html   # 微國家百科 - 全球微國家資訊
├── document-manager.html   # 文件管理系統 - 管理國家法規
├── styles.css              # 全局樣式表
├── script.js               # 主頁面互動功能
└── README.md               # 本說明文件
```

## 功能說明

### 1. 主頁面 (index.html)
- 資優民主國首頁
- 歷史編年史展示
- 憲法全文
- 法規列表與查看
- 政府組織
- 國家儀表板

### 2. 微國家百科 (micronation-wiki.html)
- 微國家定義與介紹
- 全球知名微國家卡片
- 微國家歷史發展
- 微國家分類
- 微國家比較表
- 資優民主國定位

### 3. 文件管理系統 (document-manager.html)
- 憲法管理
- 法規編輯與儲存
- 文件預覽
- 文件匯出
- 文件搜尋
